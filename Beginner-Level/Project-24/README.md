## ğŸ”‘ Key Typography Concepts

### Font Classification
- **Serif** â€” Has small strokes at letter ends (Playfair, Merriweather, Lora)
- **Sans-Serif** â€” Clean, no extra strokes (Inter, Montserrat, Poppins)
- **Monospace** â€” Every character has the same width (Roboto Mono)
- **Display** â€” Designed for large sizes and impact (Bebas Neue)

### Weight Contrast
Pairing a heavy heading weight (700â€“900) with a regular body weight (300â€“400) creates natural hierarchy without changing the font family.

### Italic vs. Oblique
- **Italic** â€” A redesigned, slanted version of the font (true italic)
- **Oblique** â€” A mechanically slanted version (synthetic)
Always prefer true italic when available.

### Letter-Spacing (Tracking)
- **Negative** (`-0.02em`) â€” Tighten large display headlines
- **Zero** â€” Default for body text
- **Positive** (`0.1em`+) â€” Use on small uppercase labels only

### The `clamp()` Function for Fluid Type
```css
font-size: clamp(min, preferred, max);

/* Example: scales between 1.8rem and 3rem */
font-size: clamp(1.8rem, 3.2vw, 3rem);
```

## ğŸ¨ Customization Guide

### Swapping Fonts
Replace the Google Fonts URL in the `<link>` tag and update `font-family` values in the HTML inline styles or CSS.

### Changing the Color Theme
Edit the CSS custom properties in `:root`:
```css
:root {
  --bg-dark: #0f1117;       /* Page background */
  --bg-card: #1a1d27;       /* Card background */
  --accent: #7c6ef5;        /* Accent / highlight color */
  --text-primary: #f0f0f5;  /* Main text */
  --text-secondary: #8889a0;/* Body text */
  --text-muted: #565870;    /* Captions */
  --border: #2c2f3e;        /* Card borders */
}
```

### Adding a New Pairing
Copy an existing `.pair-card` block inside the `#pairings` section, then update the heading font, body font, demo text, and tags.

### Adding a New Specimen
Copy an existing `.specimen-card` block inside `#specimens` and replace the font family, weights, and sample text.
