A reusable modal/overlay component built with vanilla JavaScript.
Part of the Code Odysseys series - JavaScript fundamentals practice.

## Concepts Covered
- Show/hide logic using CSS classes toggled via JavaScript
- Event handling: click, keydown
- Event propagation and `stopPropagation`
- Detecting clicks on the overlay vs. the modal itself (`event.target`)
- Keyboard accessibility (closing with the Escape key)
- Locking body scroll while a modal is open

## Features
- Open modal via button click
- Close modal via close button, "Got it" button, clicking outside the modal, or pressing Escape
- Smooth fade and slide transition
- Background scroll disabled while modal is open

Open `index.html` in any browser. No build step or dependencies required.
