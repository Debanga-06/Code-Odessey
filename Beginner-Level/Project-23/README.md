## üåà Color Schemes Explained

### 1. Monochromatic
**Theory**: Uses variations of a single hue  
**Best For**: Elegant, cohesive designs; minimalist interfaces  
**Use Case**: Background variations, subtle hierarchies  
**Example**: Different shades of blue for a professional website

### 2. Analogous
**Theory**: Colors adjacent on the color wheel (¬±30¬∞, ¬±60¬∞)  
**Best For**: Harmonious, serene designs; nature-inspired palettes  
**Use Case**: Gradients, gentle color transitions  
**Example**: Yellow-orange-red for a warm autumn theme

### 3. Complementary
**Theory**: Colors opposite on the color wheel (180¬∞)  
**Best For**: High contrast, vibrant designs; call-to-action elements  
**Use Case**: Primary vs. accent colors  
**Example**: Blue and orange for sports branding

### 4. Triadic
**Theory**: Three colors equally spaced (120¬∞ apart)  
**Best For**: Vibrant yet balanced designs; playful interfaces  
**Use Case**: Multi-category systems, data visualization  
**Example**: Primary colors (red, yellow, blue)

### 5. Tetradic (Square)
**Theory**: Four colors equally spaced (90¬∞ apart)  
**Best For**: Rich, complex designs; maximum color variety  
**Use Case**: Dashboard UIs, infographics  
**Example**: Multi-faceted brand systems

### 6. Split-Complementary
**Theory**: Base color + two colors adjacent to its complement  
**Best For**: High contrast with more nuance; sophisticated designs  
**Use Case**: UI with primary and multiple accent colors  
**Example**: Blue with yellow-orange and red-orange

### Lock Mechanism
```javascript
let lockedColors = new Set();

function toggleLock(index) {
    if (lockedColors.has(index)) {
        lockedColors.delete(index);
    } else {
        lockedColors.add(index);
    }
}

// During generation
palette = colors.map((color, index) => {
    if (lockedColors.has(index) && palette[index]) {
        return palette[index]; // Keep locked color
    }
    return color; // Use new color
});
```
