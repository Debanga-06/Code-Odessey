# 053 - Simple Carousel

A basic content carousel built with vanilla JavaScript, HTML, and CSS.
Part of the Code Odysseys series - JavaScript fundamentals practice.

## Concepts Covered
- DOM selection and manipulation
- CSS transforms (`translateX`) and transitions for sliding motion
- Event listeners (click)
- `setInterval` / `clearInterval` for autoplay behavior
- Dynamic element creation (dot indicators generated in a loop)
- Modulo operator for circular navigation

## Features
- Next / Previous navigation buttons
- Clickable dot indicators showing the active slide
- Autoplay every 3 seconds
- Autoplay timer resets whenever the user navigates manually

Open `index.html` in any browser. No build step or dependencies required.
